# DisinfoTrack

A toolkit for tracking and analyzing disinformation.

## Installation

```bash
pip install -e .
```

## Project Structure

- `disinfotrack/collection/` - Data collection utilities
- `disinfotrack/preprocessing/` - Data preprocessing and cleaning
- `disinfotrack/analysis/` - Analysis modules
- `disinfotrack/visualization/` - Visualization utilities
- `disinfotrack/evaluation/` - Evaluation metrics and tools
